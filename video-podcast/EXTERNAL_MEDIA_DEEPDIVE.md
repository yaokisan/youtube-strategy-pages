# External Media Deep Dive

Collected: 2026-05-24
Preset: video-podcast

This file strengthens the non-YouTube side of the benchmark. The goal is not to copy the look of these sources, but to borrow the viewing attitude and compress it into a YouTube thumbnail.

## Strongest Borrowable Moves

1. **Podcast配信棚/カバーアート**
   番組認識とシリーズ感だけ借りる。YouTubeではロゴ中心にしない。

2. **配信番組/トークキービジュアル**
   対話を録音風景ではなく、見る価値のある番組として置く。

3. **Webインタビューシリーズ**
   質問装置、本人反応、場所性を作り、マイク以外で視線を止める。

4. **雑誌/人物インタビュー**
   ゲストを単なる話者ではなく、語るに値する人物として見せる。

5. **Podcast業界/制作知見**
   カバーアートとYouTubeサムネの役割差を明確にする。
